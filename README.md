## 📖 简介 (Introduction)

**企数睿思 (QEESHU RUISI)** 是专为咨询行业打造的软硬一体化“超级助理”原型系统。
本项目展示了其基于 React 构建的高保真前端 GUI 界面，旨在将咨询公司海量的历史资料（PDF、PPT、访谈录音）转化为触手可及、即时响应的生产力工具。

**QEESHU RUISI** is a hardware-software integrated "Super Assistant" prototype tailored for the consulting industry. 
This repository showcases its high-fidelity frontend GUI built with React, designed to transform a consulting firm's massive historical assets (PDFs, PPTs, interview recordings) into accessible, instantly responsive productivity tools.

## ✨ 核心模块 (Core Features)

### 1. 智能工作台 (Intelligent Dashboard)
全局概览。展示边缘计算终端的算力状态、活跃项目进度，以及 AI 主动推送的高价值推荐和客户经营健康度预警。
*Global overview displaying edge-compute status, active project progress, proactive AI-driven knowledge recommendations, and client health alerts.*

### 2. 知识库诊断检索 (Smart Search & Diagnosis)
不仅是检索工具，更是交互式洞察面板。针对命中的历史案例，AI 会自动提取核心观点、推荐关键图表，并支持自然语言追问（AI Chat）。
*More than a search tool—it's an interactive insight panel. For matched historical cases, the AI extracts core concepts, recommends charts, and supports natural language follow-up questions.*

### 3. AI 撰写助手 (Advanced AI Writer)
引入类似 Notion 的多模态 Block 编辑器。
* **方法论审查**：实时检测当前报告是否缺失标准模型框架，并支持“一键 AI 补齐”。
* **动态图表生成**：自动将数据转化为专业的 React 图表（如面积图）。
* **内联 AI 组件**：输入 `/` 唤出智能生成菜单。
*A Notion-like multimodal block editor. Includes real-time methodology compliance checks with "one-click AI completion", dynamic chart generation, and inline slash commands for AI generation.*

### 4. 融合客户管理 (Smart CRM)
聚焦于客户的“经营健康度”。AI 会根据业务数据和历史项目履约情况，动态给出诊断提示，并在客户评分骤降时生成专属预警和处置建议。
*Focused on client "Operational Health". AI dynamically provides diagnostic tips based on business data and historical project performance, generating specific alerts and action plans when a client's score drops.*

### 5. 本地资产与算力管理 (Assets & Hardware)
管理 Qee-Box 边缘 AI 终端。可视化监控 CPU/NPU 的占用率波动，以及展示智力资产向量化引擎（多模态解析）的实时进度队列。
*Management for the Qee-Box edge AI terminal. Visually monitors CPU/NPU utilization and displays the real-time processing queue of the intellectual asset vectorization engine.*

### 6. 系统与方法论设置 (Methodology Settings)
* **思考策略配置**：自由切换 智能混合模式 / 严格私有模式 / 通用公开模式。
* **预置全球框架库**：一键激活 麦肯锡 7S、波士顿矩阵、波特五力模型等经典理论作为诊断参考系。
* **私有 IP 管理**：上传咨询公司自有的核心方法论。
*Configure thinking strategies, activate built-in global frameworks (McKinsey 7S, BCG Matrix, Porter's Five Forces), and manage proprietary intellectual property (IP).*

## 🚀 快速启动 (Quick Start)
